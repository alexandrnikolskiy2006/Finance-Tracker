---Money Lover---
Является приложением для отслеживания расходов, долгов, установки бюджетов и сбережений.
особенности:
+ прогноз будущих трат на основе вашей истории
+ советы для следования бюджету пользователя
+ предоставление быстрых и понятных отчетов
+ система безопасности (PIN-код/отпечаток пальца)
+ синхронизация на все мобильные устройства
+ настройка повторяющихся транзакций
+ использование нескольких валют и счетов
+ встроенный калькулятор
+ подключение банковских счетов
++ несколько кошельков
++ несколько бюджетов
++ разнообразные формы отчетов
++ экспорт в Google Sheets

недостатки:
- донат
- ошибки синхронизации
- не всегда точные прогнозы
- реклама



---Monefy---
Приложение, которое поможет спланировать бюджет и вести контроль расходов
особенности:
+ простой и понятный интерфейс
+ безопасная синхронизация через учетную запись Google Drive
+ поддержка различных валют
+ управление категориями
+ система безопасности ( пароль )
+ встроенный калькулятор
+ отсутствие рекламы
++ можно создавать свои категории
++ синхронизация данных между всеми устройствами

недостатки:
- донат
- нет синхронизации
- запрашивает данные о локации пользователя


---Денежный менеджер---
Бухгалтерское приложение для управления капиталом и отслеживания затрат
особенности:
+ возможность записывать регулярные траты (подписки)
+ вывод статистики в приложении
+ предоставление отчета пользователю
+ многоуровневые категории
+ привязка транзакций по дням (возможность просматривать через встроенный календарь)
++ система безопасности (отпечаток пальца)
++ неограниченное кол-во повторяющихся трат 

недостатки:
- сложный интерфейс
- донат
- задержки обновлений
